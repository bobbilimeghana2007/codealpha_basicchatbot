# Basic Chatbot - CodeAlpha Internship Task

## 📌 Project Description
This is a simple rule-based chatbot developed using Python.

The chatbot interacts with users using predefined responses and also saves chat history into a text file.

## 🚀 Features
- Greeting responses
- Joke feature
- Motivation messages
- Current time display
- Chat history saving
- Exit command

## 🛠 Technologies Used
- Python
- Loops
- Conditional Statements
- File Handling
- Datetime Module

## ▶ How to Run
1. Open terminal
2. Run:
python chatbot.py
